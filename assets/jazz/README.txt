Jazz rhythm-section icons
=========================

Drop your own instrument icons in THIS folder to replace the built-in line
drawings on the jazz rotation board (Jazz tab, the agenda Rhythm Section panel,
and Present mode).

Use these exact file names (PNG, ideally square, transparent background):

    drums.png        (drum set)
    guitar.png
    bass.png         (string / electric bass)
    vibraphone.png   (also used for marimba / bells / other mallets)
    piano.png        (also used for electric piano / keys)

Any size works — they're scaled down automatically. If a file is missing, a
simple built-in drawing is used instead, so the board always shows an icon.
